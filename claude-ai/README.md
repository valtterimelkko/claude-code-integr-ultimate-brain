# Claude.ai Integration with n8n

This folder contains resources for integrating Ultimate Brain with **Claude.ai** (web-based Claude) through **n8n** workflow automation.

## What's Inside

- **`n8n-workflows/`** - n8n workflow files and documentation for Claude.ai integration

## Overview

Claude.ai has built-in integration with n8n, allowing you to:
- Create automated workflows that interact with your Ultimate Brain
- Use Claude.ai to orchestrate tasks across your Notion workspace
- Build complex automations without leaving the Claude.ai interface

## Why Use This Approach?

- **No installation required** - Everything runs on the web
- **Collaborative workflows** - Share workflows with team members
- **Visual workflow builder** - Design automations graphically
- **Scheduled execution** - Run tasks on a schedule or trigger them manually
- **Easy to modify** - Update workflows without touching code

## Getting Started

See the [n8n-workflows/README.md](./n8n-workflows/README.md) for detailed setup instructions and workflow examples.

## Comparison: Claude Code vs Claude.ai Integration

| Feature | Claude Code/Desktop | Claude.ai + n8n |
|---------|---------------------|-----------------|
| Setup Required | Yes (installation) | No (web-based) |
| Collaboration | Personal use | Team workflows |
| Real-time Chat | Limited integration | Full integration |
| Scheduled Tasks | Manual triggering | Built-in scheduling |
| Maintenance | Local configuration | Cloud-based |
| Cost | Included with Claude subscription | n8n free tier available |

Choose the integration method that best fits your workflow!
